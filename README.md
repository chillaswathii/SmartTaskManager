# Smart Task Management System

A Java and MySQL based Smart Task Management System developed using JDBC connectivity.

## Features

* Add Tasks
* Display Tasks
* Search Tasks
* Delete Tasks
* Complete Tasks
* Task Analytics
* Priority-based Task Execution
* MySQL Database Integration

## Technologies Used

* Java
* JDBC
* MySQL
* SQL
* Git & GitHub

## Project Structure

* `Main.java` → Main menu and execution
* `Task.java` → Task model class
* `TaskManager.java` → Task operations
* `DBConnection.java` → Database connectivity

## Database Used

MySQL Database:

* Database Name: `taskmanager`
* Table Name: `tasks`

## How to Run

Compile the project:

```bash
javac *.java
```

Run the project:

```bash
java -cp ".;lib/mysql-connector-j-9.7.0.jar" Main
```
